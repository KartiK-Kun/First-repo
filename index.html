git clone : used to clone repo on your your local machine. git status : It is used to check the status of code


Git status
M(modified) : It represents the file has been modified.
U(Untracked): this means that git doesn't yet track e.g when we create a new folder or file in our local machine.
staged: It means that we have added the changes and these changes are ready to be commited.
Unmodified: unchanged.

add and commit command


git add : add new or changed files in your working directory to the git staging area.
And if we want to add all the changes in one go then we can use git add .
git commit -m "message regarding the commit"


Push command


git push origin main: upload local repo content to remote repo
